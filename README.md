# VersionControlGitApp

Tato Desktopová aplikace slouží k utřídění a správě lokálních i vzdálených repozitářů, které se nacházející na platformě Github.
S její pomocí je možné provádět základní úkony obsažené ve verzovacím nástroji Git. 
Jejím účelem je zjednodušení práce na projektech. 
Součástí  je záložka statistik. V ní se zobrazují lidé, kteří přispěli k vývoji projektu a další statistiky ohledně commitů a větví.



# Získání tokenu
Pro získání tokenu je nutné jít na stránku https://github.com/settings/tokens
Zde dáte vytvořit nový token, zaškrtnete veškerá práva pro aplikaci.
Tento token následně vložíte přímo do aplikace.
